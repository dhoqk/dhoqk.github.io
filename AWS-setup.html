AWS EC2 Setup + node.js

Create Amazon Web Services(AWS) EC2 instance

General Outline

  • Sign-up for free AWS account
  • Follow guidelines (Steps 1 to 5) here
  • Follow steps(1-3) here
  • Open command line tool and connect to your instance
  • Create filesystems on new device and mount it
  • Install node.js and mongodb
  • Configure system to save data to new EBS

Installing node.js

If you have followed general outlines up to step 3, then you should have your terminal open. If you haven't please do so now.

Next steps are from Kostas MAVROPALIAS blog (link at the bottom). Getting packages necessary for compiling node.js and clone from git. In terminal:



        sudo yum install gcc-c++ make
        sudo yum install openssl-devel
        sudo yum install git
        git clone git://github.com/joyent/node.git
        cd node

After installing node.js we are installing Express for web framework. As of today, Jan 16, 2015, most current and stable version of node.js was v0.10.9.

Side Note


Type 'git tag -l' to see list of all versions. If there are later versions just replace the version number following 'checkout' command. Node.js follows versioning schema of "even numbers are stable". So, v0.10.x are stable not v0.11.x In terminal:



        git checkout v0.10.9
        ./configure
        make
        sudo make install

If your are familiar with Vi text editor:


        sudo vi /etc/sudoers
        #find the line that has:
        #Defaults    secure_path = /sbin:/bin:/usr/sbin:/usr/bin
        #and add at the end. After '/urs/bin'
            :/usr/local/bin

If not use nano text editor.


        sudo nano /etc/sudoers
        #it should open up text editor within terminal
        #use arrow keys to navigate to line that has:
        Defaults    secure_path = /sbin:/bin:/usr/sbin:/usr/bin
        #and add the follwoing at the end of the line after "/usr/bin"
            :/usr/local/bin
        #hit control x (to save and exit)
        #overwrite? yes
        #file name sudoers? enter

Next we need to install node package manager NPM.



        git clone https://github.com/isaacs/npm.git
        cd npm
        sudo make install

If it gives you an error at "sudo make install" try again to force install. Or try using sudo make install -g


And that's it for installing node.js and npm. However, you still need to install Express and mongoDB on EBS volume.
